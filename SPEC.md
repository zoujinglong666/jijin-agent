- **更新日期**: 2026-05-11

### 平台与技术

## 3. 功能规范

### 3.1 首页 Dashboard（核心）

#### 模块 1: 今日风险提示
- **功能**: 展示当前持仓的风险提醒
- **规则**: 最多显示 3 条，按风险等级排序（高 > 中 > 低）
- **用户操作**: 查看原因 / 我知道了 / 稍后提醒
- **数据来源**: 本地规则引擎

#### 模块 2: 情绪风险指数
- **功能**: 量化当前仓位超出心理承受范围的程度
- **范围**: 0-100
- **等级划分**:
  - 0-30: 稳健
  - 31-60: 中性
  - 61-80: 激进
  - 81-100: 高风险
- **计算权重**:
  - 单一板块集中度: 35%
  - 高波动资产占比: 30%
  - 重复主题基金数量: 20%
  - 最近频繁查看账户: 15%

#### 模块 3: 仓位暴露图
- **功能**: 按主题展示持仓分布
- **主题列表**: 半导体、AI、新能源、医药、消费、红利、债券、黄金、港股、美股、现金、其他
- **展示**: 水平条形图 + 百分比

#### 模块 4: 场景压力测试
- **功能**: 模拟市场下跌对账户的影响
- **预设场景**:
  - 半导体下跌 5%
  - 新能源下跌 8%
  - 市场整体回撤 10%
- **输出**: 预计账户回撤百分比 + 约亏损金额

#### 模块 5: 最近减仓记录
- **功能**: 展示最近一次的减仓计划记录
- **格式**: 日期 + 基金名称 + 减仓比例

### 3.2 持仓管理

#### 添加基金
- **必填字段**:
  - 基金名称
  - 基金代码
  - 持有金额
  - 当前收益率（%）
  - 所属主题（手动选择）

#### 编辑基金
- 支持修改所有字段
- 支持删除基金

#### 基金列表
- 按持有金额降序排列
- 显示基金名称、代码、金额、收益率、所属主题

### 3.3 基金详情页

- 展示基金完整信息
- 显示仓位占比（占总投资金额）
- 提供"记录减仓计划"功能

### 3.4 模拟减仓系统

- 输入减仓比例（0-100%）
- 生成参考金额
- 本地记录减仓计划
- **重要提示**: "该记录仅用于仓位管理，不会执行真实交易"

### 3.5 历史记录页

- 减仓记录列表
- 风险指数变化趋势图
- 情绪风险变化

### 3.6 设置页

- 风险偏好配置（保守/平衡/激进）
- 数据管理（导出/清除）
- 关于产品
- 风险声明

## 4. 规则引擎

### 4.1 JSON 规则配置
```json
{
  "version": "1.0",
  "rules": [
    {
      "id": "risk_001",
      "name": "高集中度提醒",
      "condition": "sector_ratio > 40",
      "level": "high",
      "title": "仓位集中度较高",
      "content": "你的单一主题仓位已经超过40%，短期波动可能明显放大。"
    },
    {
      "id": "risk_002",
      "name": "重复主题提醒",
      "condition": "same_sector_fund_count >= 3",
      "level": "medium",
      "title": "重复主题较多",
      "content": "你持有多只同主题基金，它们可能存在较高重叠。"
    },
    {
      "id": "risk_003",
      "name": "高波动提醒",
      "condition": "high_volatility_ratio > 70",
      "level": "high",
      "title": "高波动资产占比较高",
      "content": "当前账户波动风险偏高，请关注仓位承受能力。"
    }
  ]
}
```

### 4.2 规则输出原则
- ❌ 禁止: "建议买入"、"建议卖出"
- ✅ 正确: 风险提醒、波动提醒、仓位提示

## 5. 数据库设计

### 表 1: funds（基金表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| name | TEXT | 基金名称 |
| code | TEXT | 基金代码 |
| amount | REAL | 持有金额 |
| profit_rate | REAL | 当前收益率 (%) |
| sector | TEXT | 所属主题 |
| create_time | INTEGER | 创建时间戳 |
| update_time | INTEGER | 更新时间戳 |

### 表 2: actions（操作记录表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 (自增) |
| fund_id | TEXT | 基金 ID |
| action_type | TEXT | 操作类型 |
| ratio | REAL | 减仓比例 (%) |
| reason | TEXT | 备注原因 |
| create_time | INTEGER | 创建时间戳 |

### 表 3: risk_history（风险历史表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 (自增) |
| risk_score | INTEGER | 情绪风险指数 |
| snapshot_json | TEXT | 快照 JSON |
| create_time | INTEGER | 创建时间戳 |

## 6. UI/UX 规范

### 6.1 设计原则
- **关键词**: 冷静、克制、非金融营销化、不刺激交易、不制造 FOMO
- **避免**: 大红大绿、暴涨暴跌动画、"立即卖出"、"暴赚"
- **推荐**: 莫兰迪色、柔和卡片、低刺激 UI、极简图表

### 6.2 色彩系统
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #6B7FD7 | 靛蓝蓝 |
| 次色 | #A5B4FC | 浅紫蓝 |
| 强调色 | #F59E0B | 琥珀色（警告） |
| 背景色 | #F8FAFC | 雪白 |
| 卡片色 | #FFFFFF | 纯白 |
| 文本主色 | #1E293B | 深灰蓝 |
| 文本次色 | #64748B | 中灰 |
| 高风险 | #EF4444 | 红色 |
| 中风险 | #F59E0B | 橙色 |
| 低风险 | #22C55E | 绿色 |
| 稳健 | #6B7FD7 | 蓝色 |
| 中性 | #A855F7 | 紫色 |
| 激进 | #F97316 | 橙色 |
| 高风险级 | #EF4444 | 红色 |

### 6.3 字体规范
- **标题**: 18-24sp, FontWeight.w600
- **正文**: 14-16sp, FontWeight.w400
- **辅助文字**: 12sp, FontWeight.w300

### 6.4 间距规范
- **页面边距**: 16dp
- **卡片间距**: 12dp
- **元素间距**: 8dp
- **卡片圆角**: 16dp

### 6.5 组件规范
- **卡片**: 白色背景、16dp 圆角、轻微阴影
- **按钮**: 主按钮（填充）、次按钮（描边）、文字按钮
- **输入框**: 圆角边框、轻微背景色

## 8. 合规要求

### 风险声明
> 本产品仅提供基金持仓分析、风险提示与行为记录功能。
>
> 所有内容不构成投资建议，不承诺收益，不提供任何证券基金交易服务。

### 产品边界
- ✅ 仓位分析、风险暴露计算、情绪风险评估、减仓计划记录、波动压力测试、行为提醒
- ❌ 不预测涨跌、不推荐买卖、不提供收益承诺、不做自动交易、不做量化策略、不做社区喊单、不做跟单
